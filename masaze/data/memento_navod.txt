NÁVOD – IMPORT DO MEMENTO DATABASE
=====================================

1. INSTALACE
   • Stáhni Memento Database z Google Play nebo App Store
   • Je zdarma (základní verze plně postačuje)

2. VYTVOŘENÍ KNIHOVNY
   • Otevři aplikaci
   • Klikni na "+" (nová knihovna)
   • Název: "Maséři ČR"
   • Vyber prázdnou šablonu

3. IMPORT CSV
   • V knihovně klikni na ⋮ (tři tečky) vpravo nahoře
   • Vyber "Import"
   • Vyber soubor: masaze_memento.csv
   • Potvrď mapování sloupců (vše by mělo sedět automaticky)
   • Klikni "Importovat"

4. POLE V DATABÁZI
   Po importu budeš mít tato pole:
   ┌─────────────────────┬────────────────────────────────┐
   │ Pole                │ Popis                          │
   ├─────────────────────┼────────────────────────────────┤
   │ Kraj                │ Název kraje (předvyplněno)     │
   │ Okres               │ Název okresu (předvyplněno)    │
   │ Město               │ Konkrétní město v okrese       │
   │ Jméno maséra        │ Celé jméno                     │
   │ Telefon             │ Kontaktní telefon              │
   │ E-mail              │ Kontaktní e-mail               │
   │ Typ                 │ Já / Kolega / Externí          │
   │ Dovednosti          │ Co masér umí                   │
   │ Cena od (Kč)        │ Minimální cena                 │
   │ Cena do (Kč)        │ Maximální cena                 │
   │ Délka masáže (min)  │ Délka ošetření                 │
   │ Mobilní masér       │ Ano / Ne                       │
   │ Poznámka            │ Cokoliv dalšího                │
   └─────────────────────┴────────────────────────────────┘

5. DOPORUČENÉ TYPY POLÍ V MEMENTO
   • Kraj          → Text (jednořádkový)
   • Okres         → Text (jednořádkový)
   • Město         → Text (jednořádkový)
   • Jméno maséra  → Text (jednořádkový)
   • Telefon       → Telefon
   • E-mail        → E-mail
   • Typ           → Výběr (Já / Kolega / Externí)
   • Dovednosti    → Text (víceřádkový) nebo Tagy
   • Cena od/do    → Číslo (celé)
   • Délka masáže  → Číslo (celé)
   • Mobilní masér → Zaškrtávací pole (Ano/Ne)
   • Poznámka      → Text (víceřádkový)
