## 📚 Historie klíčových lekcí

1. **Template literals = ZAKÁZANÉ** → vždy string concatenation
2. **`${...}` v JS stringu** = crash i když Node nehlásí chybu
3. **Extra `<script>` tagy** = bordel/fronta mají vlastní script
4. **Duplicitní deklarace** (G, sw) = `const` z bordel crashuje
5. **Div balance** = vždy kontrolovat po každé změně
6. **Views = 56, ALL_TABS = 55** = správný stav
7. **Záloha VŽDY** před každou změnou!
